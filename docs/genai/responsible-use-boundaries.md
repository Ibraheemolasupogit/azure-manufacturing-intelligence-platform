# Responsible Use Boundaries

The assistant is a deterministic local simulation over synthetic evidence. It is appropriate for portfolio demonstration, governance testing, and later dashboard narrative inputs.

It must not be used as a live operations assistant, safety authority, maintenance procedure generator, customer or supplier intelligence source, or proof of real-world plant status. It does not contain real customer, supplier, employee, plant, product, or commercial operations data.

External model calls are disabled by configuration and validated in outputs.
