# Deterministic Response Synthesis

Responses are generated by local rules over retrieved evidence. The synthesizer uses evidence titles, paths, row counts, hashes, and key metrics to produce concise grounded answers with citations.

Supported tasks are executive operations brief, forecast inventory risk, production variance explanation, quality alerts, maintenance risks, platform health, data lineage, and data quality governance status.

The output never claims real operational truth and never claims Azure AI services were used.
