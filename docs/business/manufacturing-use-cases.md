# Manufacturing Use Cases

The following use cases are planned for later milestones and are not implemented in Milestone 1.

| Use case | Intended value |
| --- | --- |
| Demand forecasting | Estimate future product demand to inform capacity and inventory planning. |
| Inventory optimisation | Identify stockout risk, overstock, safety-stock coverage, and replenishment priorities. |
| Production planning | Compare actual production against plan and surface throughput constraints. |
| Quality anomaly detection | Detect unusual quality measurements, defect patterns, and deteriorating batches. |
| Predictive maintenance | Identify machines that require inspection or maintenance based on health signals. |
| Supplier-risk intelligence | Evaluate delivery reliability, in-full performance, quality score, and defect risk. |
| Warehouse-flow analytics | Analyse material movement patterns and operational bottlenecks. |
| Operational recommendation generation | Use governed context to draft recommended actions for human review. |
