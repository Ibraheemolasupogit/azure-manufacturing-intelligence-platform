# Dashboard Lineage And Manifest

The dashboard manifest records run identity, configuration hash, input manifests, input hashes, output files, row counts, sizes, SHA-256 hashes, semantic model summary, page count, visual count, metric count, validation status, synthetic classification, deployment flags, Git commit, and upstream immutability confirmation.

Lineage links governed input evidence to dimensions, facts, scorecard, metric catalogue, semantic metadata, page and visual specs, reports, manifest, and lineage outputs. It does not claim Microsoft Purview registration.
