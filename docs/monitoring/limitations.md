# Monitoring Limitations

Monitoring is deterministic local evidence monitoring over synthetic portfolio files. It does not ingest live logs, stream telemetry, deploy Azure resources, use Azure SDKs, publish dashboards, or provide formal operational SLAs.

Health scores are heuristic indicators for portfolio observability. They distinguish operational health, data-quality health, and analytics-result health, but they are not production service-level measurements.

Milestone 9 GenAI operations assistant work is deliberately deferred.
