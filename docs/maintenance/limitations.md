# Maintenance Limitations

The pipeline uses synthetic portfolio data only. It does not represent real plants, machines, sensors, suppliers, customers, employees, safety systems, or commercial operations.

Failure-risk scores are deterministic heuristics, not calibrated failure probabilities. Isolation Forest scores are relative diagnostics. Runtime/service indicators are proxies because explicit maintenance schedules are unavailable. Investigation context is not root-cause proof.

Milestone 8 monitoring is deliberately deferred. No live telemetry, Azure Monitor integration, dashboards, GenAI, cloud deployment, databases, or streaming services are implemented.
